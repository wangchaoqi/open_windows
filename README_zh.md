# Window Switcher — 窗口随机切换器

轻量级 Windows 工具，定时随机切换打开的软件窗口，模拟 Alt+Tab 效果。人离开电脑时让屏幕看起来像有人在操作。

> 🟢 **纯 Python 标准库实现，零外部依赖，无需 pip 安装。**

## 功能特性

- **随机窗口切换** — 在可见窗口之间随机切换，间隔可配
- **连切模式** — 偶尔快速连切 2–3 次，模拟真实的 Alt+Tab
- **空闲检测** — 检测到鼠标/键盘有操作时自动跳过切换
- **每日自动停止** — 到达设定时间（默认 18:00）自动停止
- **系统托盘图标** — 绿色运行中 / 灰色已停止，直观明了
- **单实例运行** — 防止重复启动多个进程
- **静默后台运行** — 正常启动无控制台窗口

## 使用说明

### 托盘图标操作
| 操作 | 效果 |
|---|---|
| **左键点击** 图标 | 立即切换一次窗口 |
| **右键点击** 图标 | 弹出控制菜单 |

### 右键菜单
| 菜单项 | 功能 |
|---|---|
| ▶ 开始切换 | 启动定时随机切换 |
| ⏹ 停止切换 | 暂停切换 |
| 🔄 立即切换 | 手动触发一次切换 |
| ⚙ 设置... | 配置间隔、空闲检测、自动停止 |
| ❌ 退出 | 退出程序 |

### 图标含义
- 🟢 **绿色** — 切换运行中
- ⚪ **灰色** — 已停止

## 配置参数

配置文件 `switcher_config.json` 与脚本在同一目录，可通过设置菜单修改或手动编辑。

| 参数 | 默认值 | 范围 | 说明 |
|---|---|---|---|
| `min_interval` | 120 | 10–3600 | 两次切换之间的最短秒数 |
| `max_interval` | 300 | 10–3600 | 两次切换之间的最长秒数 |
| `burst_enabled` | true | true/false | 是否启用连切模式 |
| `burst_chance` | 0.15 | 0–1 | 每次触发连切的概率 |
| `idle_threshold` | 60 | 0–600 | 用户 N 秒内有操作则跳过切换（0=关闭） |
| `auto_stop_time` | "18:00" | "HH:MM" 或 "" | 每日自动停止时间（空=关闭） |

### 配置示例
```json
{
  "min_interval": 120,
  "max_interval": 300,
  "burst_enabled": true,
  "burst_chance": 0.15,
  "idle_threshold": 60,
  "auto_stop_time": "18:00"
}
```

## 工作原理

### 窗口切换
- 通过 `EnumWindows` 枚举所有可见顶层窗口
- 自动过滤不可见窗口、UWP 隐藏窗口、空标题窗口
- 使用 `AttachThreadInput` 绕过 Windows 前台锁定限制
- 随机选择一个目标窗口并切换到前台

### 空闲检测
- 通过 `GetLastInputInfo` 获取距离上次键盘/鼠标操作的时间
- 用户最近有操作则跳过本次切换，避免打扰正常使用
- 只在人真正离开时才执行切换

### 自动停止
- 每次切换前检查系统时间
- 到达设定时间自动停止，当天不再触发
- 第二天自动重置，按时再次停止

## 常见问题

**问：启动后看不到托盘图标？**
答：图标可能被折叠在任务栏溢出区。点击任务栏右下角的 `^` 箭头，找到图标后拖到可见区域。

**问：会触发杀毒软件吗？**
答：不会。程序只使用标准 Windows API，不含编译二进制、不下载文件、不修改系统。

**问：怎么调整切换频率？**
答：右键托盘图标 → 设置，或直接编辑 `switcher_config.json`。

**问：能改成下午 5 点自动停吗？**
答：可以。打开设置把自动停止时间改成 `17:00`，或直接改配置文件。

**问：支持 Windows 11 吗？**
答：Windows 10 和 Windows 11 全部支持。

**问：需要安装什么依赖吗？**
答：不需要。只要有 Python 3.8+ 就能跑，全部使用 Python 自带的标准库。
